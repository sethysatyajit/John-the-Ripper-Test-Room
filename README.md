# 🔐 John the Ripper · Hash Cracking Test Room

> **A fully responsive, hacker‑themed web app to practice hash identification and cracking — just like John the Ripper.**

---

## 🧠 Overview

**John the Ripper Test Room** is an interactive training platform designed for cybersecurity enthusiasts, CTF players, and aspiring penetration testers.  
It presents **50 hash challenges** (MD5, SHA1, SHA256) across **5 difficulty rooms** (Easy → Master).  
Each challenge displays the hash type and value, and for tougher hashes, a **wordlist hint** is provided — just like real‑world password cracking.

**No actual cracking happens client‑side — it's a simulation to test your recognition and dictionary skills.**

---

## ✨ Features

- 🎯 **50 Unique Hash Challenges**  
  Includes MD5, SHA1, and SHA256 hashes with varying difficulty.

- 🧩 **5 Progressive Rooms**  
  Warm‑up → Medium → Hard → Advanced → Master (10 questions each).

- 💡 **Smart Wordlist Hints**  
  For "hard" questions, a 20‑30 word wordlist appears to help you guess the plaintext.

- 📋 **Copy Hash Button**  
  One‑click copy to clipboard — ready for external tools.

- 🎨 **Hacker Aesthetic**  
  Glitch effects, scanline overlay, typewriter animation, red/green neon colors.

- 💾 **Progress Saved Locally**  
  Your solved hashes and room completions persist in browser `localStorage`.

- 📱 **Fully Responsive**  
  Works flawlessly on desktop, tablet, and mobile.

- 📝 **Answer Key & Room Summary**  
  After completing a room, review correct answers and your score.
